# 🐍 Snake Game

A classic **Snake Game** built using **Python** and the **Turtle Graphics** library. This project demonstrates object-oriented programming, collision detection, score tracking, and interactive game development.

---

## 🚀 Features

- 🎮 Smooth snake movement
- 🍎 Random food generation
- 📈 Score tracking
- 🏆 High score saving
- 💥 Collision detection with walls
- 🐍 Self-collision detection
- ⚡ Simple and responsive gameplay
- 🖥️ Easy-to-use interface

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone this repository.
2. Make sure Python 3 is installed.
3. Run:


python snake-game.py




## 🎯 Controls

- ⬆️ Up Arrow – Move Up
- ⬇️ Down Arrow – Move Down
- ⬅️ Left Arrow – Move Left
- ➡️ Right Arrow – Move Right
